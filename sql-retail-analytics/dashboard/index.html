Analytical Project Summary

This project structures a multi-dimensional Star Schema database in SQLite containing 12,500 retail sales records. The database organizes transactions around store locations, customers, products, and calendar dimensions.

The primary analytical objective is identifying promotional pricing leakages, assessing regional logistics cost strains (freight friction), and analyzing customer cohort returns.

Select the SQL Analytical Console tab to explore the queries used, inspect SQL statements, and view interactive charts generated directly from query result rows.
